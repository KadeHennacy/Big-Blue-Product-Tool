Notes about importing products from a file
  1. 1. Use this file to import products that do not have options such as size and color. Products with such options cannot be imported through this file. To add products with options such as size and color, use the Add New Product wizard
  2. 2. The file format must be CSV, comma-separated values with optional quotation marks surrounding a value
    Example: The entries Column 1,"Column 2","1234","Column,3","Column ""4"" yield these imported results:
    Column 1
    Column 2
    1234
    Column,3
    Column "4"
  3. 3. The column headers described below must appear exactly as shown here on the first row/line of the file. All headers from the first table must be in the CSV file. Any header from the second table is optional and can be skipped. The columns can be in any desired order.
  4. 4. Products will be inserted into the Store Home Page category sorted in the sequence defined in the .csv file. Category assignments and sort orders can be adjusted post import as desired.
Required Columns
Column Header Description Notes
NAME Name of the Product 200 characters.
SHORT_DESC Short Description of the Product 500 characters.
POINT_OF_SALE_DESC Point of Sale Description of the Product 500 characters.
LONG_DESC Long Description of the Product 30000 characters.
SHIP_CLASS Name of the Shipping Class for the Product Must match an existing store shipping class name ( US Mail, Priority Mail, None, test ).
PRICE Price of the Product Use the form 1.23 with no commas or dollar signs. Price cannot be less than 0.00.
PRODUCT_TYPE Type of the Product Valid product types are Generic, Digital, or Donation. If the product type is donation then the price must be 0.00.
Optional Columns
Column Header Description Notes
QTY Quantity Available Leave blank to indicate that quantity is not tracked (default is that quantity is not tracked)
BACK_ORDERABLE Is Product back orderable? Set to Y or Yes if product is back orderable, otherwise N or No (default is No)
STOCK_NO Stock # If left blank, a random number will be assigned. 30 characters.
ENABLED Should Product show up as available to order? Set to Y or Yes to display the product immediately. Set to N or No to disable the product's display. (Default is Yes.)
AUTH_REQD Is authorization required to purchase this product? Set to Y or Yes if authorization is required, otherwise N or No (default is No)
AUTH_PROMPT Prompt displayed to the user if authorization is required Prompt displayed to the user if authorization is required
AUTH_PROMPT_ANS Answer to prompt from the user if authorization is required Must be filled in if authorization is required. 50 characters.
TAX_CLASS Tax Class Name Set to the appropriate tax class for this Product (default is the "default tax rate" tax class)
CREDIT_ACCOUNT Credit Accounting Code Name Retain the default Credit Accounting Code for the store's GL submissions ( ), or enter the name of another Credit Accounting Code as an override.
